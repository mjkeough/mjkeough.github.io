Solutions for Exercises

We’ve provided some brief notes and guidance on R code for most of the questions and exercises that we’ve provided.

We’ve put this material in the Instructors area, because we’ve found that when complete code is provided along with exercises, students tend to just follow the code, rather than thinking for themselves. We’ve found it a difficult balance this situation and tutorials becoming introductory coding sessions when no code is provided.

The code is helpful if it’s released a bit at a time, but that depends on the students’ familiarity with statistics and R. In practice, they proceed at wildly different rates, so even “chunk at a time” mode is difficult.

There are brief notes about what we’d expect students to answer for some of the questions. This material is mostly in two categories, explaining the analysis pathway, and drawing conclusions, including areas of uncertainty.
