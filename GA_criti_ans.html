##What (biological) question(s) were the data designed to answer?##
Broadly, how does fire affect ants. | Specifically, how do prescribed fires, tree species (oaks and hickories) and tree size (large and small) affect the presence of one particular ant species, black carpenter ants | ##What kind of statistical model(s) were fitted to the data?##

Analysis of variance models. Data collected were presence/absence of ants on individual trees. Trees were within 5 plots at each site, and data were aggregated up to % of trees with ants. These percentages were the variables analysed. This step makes sense! | Claimed in methods that analyses were one-way anova, but they also talk about interactions between factors. ##What changes would you make to the data analysis?## Tell the reader what was actually done (not clear from ms) The analysis bears no relationship to the structure of the data. The authors ignored things like collection of samples in different years and the pairing of burned and unburned sites. They also did simple anovas when the data for different tree species and sizes clearly came from the same set of plots, i.e., there were not spatially independent estimates for each species, etc. The actual analysis/design is quite complicated. Here’s my take on it: a. At a large scale, data were collected in 3 years (Y) b. In each year, there were 3 pairs (P) of sites c. Within each pair, one was burned and the other was not (B) d. Within each site data were collected on the % of trees having ants for each of 3 tree species (S) of 3 different sizes, measured as DBH (D) e. Technically, data came from 5 plots within each site, but it’s simplest to aggregate the data at site level (all tree species/sizes may not be in each 30 m plot) f. The experimental unit is the site – the scale at which fire treatments were applied. The linear model to describe this data is best treated as a repeated measures design (or partly nested). a. The “subjects” are sites i. Each site occurs in one year ii. Is one of a pair of sites differing in burning treatmentb. The between-subjects part of the design has Years, Pairs nested within years, and Burning treatments applied to pairs: Y, P(Y), B, Y*B | | | | c. The “repeated” measures are the ant presences on different species and sizes of trees, and one value for each combination comes from a site. | | | | i. The within subject measurements form a factorial array of Species and Size: S, D, S*D | | | | 5. The correct statistical model thus has 5 factors, 3 between and 2 within subjects. One of those factors (Pairs) is a random effect. The others are fixed. Years could be either, but with 3 consecutive years, it’s probably best treated as fixed. | | | | 6. The full analysis table is complicated, and needs to be evaluated carefully, with the presence of both fixed and random effects. | | | | 7.
